In GraphQLite, mutations are created [like queries](queries.mdx).

To create a mutation, you must annotate a method in a controller with the `@Mutation` annotation.

For instance:

<Tabs
  defaultValue="php8"
  values={[
    {label: 'PHP 8', value: 'php8'},
    {label: 'PHP 7', value: 'php7'},
  ]}>
  <TabItem value="php8">

```php
namespace App\Controller;

use TheCodingMachine\GraphQLite\Annotations\Mutation;

class ProductController
{
    #[Mutation]
    public function saveProduct(int $id, string $name, ?float $price = null): Product
    {
        // Some code that saves a product.
    }
}
```

  </TabItem>
  <TabItem value="php7">

```php
namespace App\Controller;

use TheCodingMachine\GraphQLite\Annotations\Mutation;

class ProductController
{
    /**
     * @Mutation
     */
    public function saveProduct(int $id, string $name, ?float $price = null): Product
    {
        // Some code that saves a product.
    }
}
```
